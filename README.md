Privacy Policy for Volume Booster

Volume Booster does not collect, store, transmit, or share any user data, browsing history, or audio content. All volume-boosting is performed entirely within your browser using the Web Audio API. The extension does not communicate with any external server. The only data it stores is your chosen volume level per open tab, kept locally via Chrome's storage.session API, and this is cleared automatically when the tab is closed.
