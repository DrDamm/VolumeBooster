Privacy Policy for Volume Booster

Last updated: 2026-07-19

Volume Booster is a Chrome extension that lets you increase the playback volume of audio and video on the current browser tab. This policy explains what data the extension does and does not handle.

Data collection

Volume Booster does not collect, transmit, sell, or share any personal data, browsing history, audio content, or website content. The extension does not communicate with any external server and has no analytics, tracking, or advertising components.

How it works

All volume boosting happens locally in your browser using the Web Audio API. The extension's content script attaches an audio gain node directly to the video/audio elements on the page you're viewing; it does not read, log, or transmit the audio or page content.

Local storage

The extension stores one thing locally on your device: the volume level you set for each currently open tab, using Chrome's storage.session API. This value never leaves your browser, is not accessible to any website, and is automatically deleted when the tab is closed.

Permissions

  • storage — remembers your chosen volume level per open tab.
  • activeTab — identifies the current tab so the popup can apply the volume change to it.
  • Host permission (all sites) — required so the volume-boosting script can run on any page you choose to boost, including embedded video players in iframes.

Changes to this policy

If this policy changes, the updated version will be posted at this same URL with a revised "Last updated" date.
